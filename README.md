# Better Call Agents

Investor-ready static website and HTML pitch deck for Better Call Agents, an agentic Legal RAG concept focused on exact citation retrieval and auditable legal workflows.

## Pages

- `index.html` - public investor showcase website
- `pitch-deck.html` - browser-based investor pitch deck
- `game.html` - Citation Runner jump-and-run with local high score

The design uses the red, yellow, black, and cream palette from the provided Legal RAG pipeline references.
